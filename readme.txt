== Installation ==

安装：
1. 在插件中搜索 <strong>胖鼠采集</strong> 安装即可
2. 将插件文件上传到/wp-content/plugins/目录 即可
3. 如果你的PHP版本小于PHP7, 请去胖鼠采集的Github下载使用胖鼠v5版本 分支名: based_php_5.6
